# Quaia: The Gaia-unWISE Quasar Catalog

It has been converted to UniverseViewer data format using the `extract.py` script.

The result has been compressed with:
```
gzip -c quaia_G20.0.dat > quaia_G20.0.gz
gzip -c quaia_G20.5.dat > quaia_G20.5.gz
```

To run the conversion script, you have to download the `quaia_G20.0.fits` and `quaia_G20.5.fits` files from https://zenodo.org/records/10403370 URL.

# Quasars and Active Galactic Nuclei (13th Ed.)

It has been converted to UniverseViewer data format using the `extract.py` script.

The result has been compressed with:
```
gzip -c catalog.dat > catalog.gz
```

To run the conversion script, you have to download the `VII_258_agn.dat.fits` or the `VII_258_qso.dat.fits` file from https://cdsarc.cds.unistra.fr/viz-bin/cat/VII/258#/browse URL.
